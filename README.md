# Offer Buy

A minimal Vue 3 + Vite landing page that highlights purchasing offers with a clean Tailwind CSS table.

## Stack
- [Vite](https://vitejs.dev/)
- [Vue 3](https://vuejs.org/) with the Composition API
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production build:
   ```bash
   npm run preview
   ```

## Styling
Tailwind is configured with a primary palette rooted at `#028e4b` and a soft surface background to keep the layout light while emphasizing the buy actions.
